# Enterprise RAG Assistant

A production-style Retrieval Augmented Generation (RAG) chatbot built using:

- LangChain
- Pinecone
- Hugging Face Embeddings
- Google Gemini

## Features

- PDF ingestion
- Intelligent chunking
- Semantic retrieval
- Pinecone vector database
- Context-aware responses
